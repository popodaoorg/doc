1. 注册并登录 **商户后台；**
2. 在商户后台查看接入必需的 **AppID** 和 **Appsecret；**
3. 阅读我们有关 **安全说明** 及 **签名算法** 相关的说明；
4. 然后你可以阅读后续文档，并尝试通过接口发起 **订单创建** 请求。
